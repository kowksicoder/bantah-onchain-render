# BXBT Utility Rules

BXBT is the BantahBro utility layer. It can be used for market creation, market boosts, premium signals, rewards, and future token launch or agent participation flows.

BantahBro should describe BXBT as utility and access, not as guaranteed profit. It should not promise token appreciation.

BXBT status is live-only. Token address, treasury address, balance, chain, market cost, boost cost, and reward amount must be read from the runtime config and wallet state.

If BXBT is not configured, BantahBro should say the utility rail is not fully configured instead of guessing.

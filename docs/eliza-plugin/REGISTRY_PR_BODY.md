## Title

`feat: add @bantah-protocol/plugin-bantah to registry`

## Body

This PR adds `@bantah-protocol/plugin-bantah` to the elizaOS plugin registry.

### Plugin

- NPM package: `@bantah-protocol/plugin-bantah`
- Repository: `github:bantah/plugin-bantah`

### What it does

This plugin gives Eliza agents Bantah prediction market actions:

- `create_market`
- `join_yes`
- `join_no`
- `read_market`
- `check_balance`

It is designed for agents that want to interact with Bantah-compatible runtime endpoints and participate in Bantah markets.

### Notes

- branding assets are included in the plugin repository
- the package includes `agentConfig`
